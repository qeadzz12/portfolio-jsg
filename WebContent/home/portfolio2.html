프로젝트 개요

프로젝트 내용: 1일 클래스 예약 사이트 세모클레스

아이디어 배경: 등록된 클래스를 보기 좋게 보여주고, 지역, 카테고리, 가격 등을 기준으로 정렬 및 검색 기능 제공

기술적 배움

  • DB 설계:
    • 효율적인 데이터 저장 및 검색을 위해 데이터베이스 구조 설계
    • 프로젝트 규모에 따라 적절한 설계가 중요하다는걸 배움
  • AJAX 활용:
    • 클래스 검색, 예약, 결제
  • 외부 API 활용:
    • 네이버 지도 API를 사용해 위치 기반 클래스 검색 기능 구현
    • 결제 및 환불 구현

협업 및 프로젝트 관리

  • 팀 소통의 중요성:
    • 다양한 기능 구현 시 팀원 간의 명확한 역할 분담과 소통의 중요성을 체감
    • 일부 기능을 담당한 팀원이 잘 되고 있다고 하다 마지막에 못하겠다고 하여 더욱 체감
  • 문제 해결 능력:
    • 각자 담당한 기능을 구현 못하는 팀원이 나와서 당황스러웠지만 그런 팀원들을 도와서 기능을 수정 및 삭제하거나 도와서 함께 완성시킴
  • 협력의 중요성:
    • 개별 작업 뿐만 아니라, 팀원의 작업을 적극 도우며 협력의 중요성을 배움

주요 기능 및 개인 기여

  • 주요 기능:
    • 클래스 목록 보기 및 검색
    • 지역, 카테고리, 가격 기준 정렬
    • 클래스 예약 및 결제
    • 네이버 지도 API를 사용한 내 주변 클래스 찾기
    • 이벤트 및 고객지원 게시판
  • 개인 기여:
    • DB 설계 및 연결
    • 클래스 검색 기능 구현
    • 클래스들이 보여지는 메인 페이지 제작
    • 원하는 클래스 위시리스트 기능 구현
    • 웹뷰 기능을 이용해 안드로이드 앱을 구현
    • 네이버 지도 API를 사용해 지도로 찾는 기능 구현
    • 결제 및 환불 기능 지원

총평

이번 프로젝트에서는 이전 프로젝트에서 경험했던 디자인 통일성 문제를 해결하기 위해 템플릿을 적극 활용했습니다. 이로 인해 프로젝트의 일관된 디자인을 유지할 수 있었습니다. 또한, 기능 구현에 어려움을 겪는 팀원이 있었지만, 팀원들이 함께 도와 문제를 해결해 나갔습니다. 이번 프로젝트를 통해 다양한 기술적 능력을 습득하고, 팀워크와 프로젝트 관리의 중요성을 깊이 깨달았습니다. 여러 도전과 경험을 통해 성장할 수 있었으며, 협업의 가치를 다시 한번 실감할 수 있었습니다.
